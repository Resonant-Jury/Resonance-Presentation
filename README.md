## 💻 本地端執行 (開發者資訊)

此網頁簡報採用無需建置 (Zero-build) 的 React 架構，您可以直接使用任何靜態檔案伺服器在本地端執行：

```sh
# 啟動伺服器 (使用 npx)
npx serve .

# 或者使用 Python 內建的伺服器
python3 -m http.server 8080
```

接著在瀏覽器中開啟 `http://localhost:8080` 即可觀看。
